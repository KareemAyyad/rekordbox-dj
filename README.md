# DropCrate

A local-first, high-performance DJ downloader and Rekordbox-ready metadata engine.

## üöÄ Quick Start (Non-Technical Users)

To start the app immediately:

- **macOS**: Double-click the `Start-macOS.command` file.
- **Windows**: Double-click the `Start-Windows.bat` file.

The first time you run it, the script will automatically:
1.  Install necessary background files (`node_modules`).
2.  Create your configuration file (`.env`).
3.  Set up your default DJ inbox (`rekordbox/DJ Library/00_INBOX`).
4.  Open the app in your browser at `http://localhost:5173`.
